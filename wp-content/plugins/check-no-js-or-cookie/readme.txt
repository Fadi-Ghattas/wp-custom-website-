== Description ==

This plugin will display a warning message to your visitors if their browser's Javascript or Cookie is disabled. You just activate the plugin and it will show the warning message to the appropriate visitor (ones who do not have JavaScript or Cookie enabled)

For detailed documentation, updates, please visit:
http://www.tipsandtricks-hq.com/

== Usage ==

There is no special usage instruction needed. Simply activate the plugin and all done.

== Installation ==

1. Unzip and Upload the folder 'check-no-js-or-cookie' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
